# README screenshots

Drop images here to have them show up in the top-level `README.md`.

Expected filenames (lowercase, PNG preferred):

| Filename        | Shown in README as     | Recommended size |
| --------------- | ---------------------- | ---------------- |
| `hero.png`      | Top hero banner        | 1600×900         |
| `tracker.png`   | Live Tracker (Mapbox)  | 1600×900         |
| `sponsors.png`  | Sponsors section       | 1600×700         |
| `admin.png`     | Admin dashboard        | 1600×900         |

Tips:

- Use PNG for crisp UI screenshots, or WEBP for smaller files.
- Keep filenames lowercase and exactly as listed above.
- 2× resolution (e.g. 3200×1800) looks great on retina displays.
